# 1次元横磁場イジングモデルの厳密対角化（L=4, h=1.0）

このリポジトリには、1次元横磁場イジングモデル

\[
H = -\sum_i \sigma_i^z\sigma_{i+1}^z - h\sum_i \sigma_i^x
\]

（周期境界条件）を厳密対角化する Python スクリプト `ising_ed.py` を追加しています。

## 実行方法

```bash
python3 ising_ed.py
```

## テストの実行方法

ユニットテストは Python 標準ライブラリの `unittest` で実行できます。追加の依存関係は不要です。

```bash
python3 -m unittest
```

特定のテストファイルだけを実行する場合は、次のように指定します。

```bash
python3 -m unittest test_ising_ed.py
```

## 出力内容

- 基底エネルギー
- 基底状態ベクトル（\(z\) 基底の順序：`|0000>, |0001>, ..., |1111>`）
- 各スピン配置（`|↑↑↑↑>` など）に対する基底状態の振幅

## 必要環境

- Python 3（標準ライブラリのみで実行可能）
